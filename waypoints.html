Waypoints

Used for marking special points on the map, to be referenced by scripting. A scenario can have up to 30 waypoints defined, each attached to a cell on the map.

Name Description
P0-5 These waypoints are used as the possible player start positions in multiplayer. In a singleplayer campaign they can be used as general-purpose waypoints.
P6-7 In singleplayer these can be used as general-purpose waypoints.
8-24 General-purpose waypoints
25 The "Flare" waypoint. Marks the location at which a drop zone flare can be created. See "DZ at Z" trigger action type. At the time of writing, the map editor erroneously displays "Flare" for waypoint #27.
26 The "Home" waypoint. Marks where the camera will begin at the start of a singleplayer scenario. At the time of writing, the map editor erroneously displays "Home" for waypoint #28.
27/Flare The "Reinf" waypoint. Marks where reinforcements will arrive by air. See "Reinforce" trigger action type and "Air" source type. At the time of writing, the map editor erroneously displays "Reinf" for waypoint #29 and "Flare" for this waypoint.
Home At the time of writing, the map editor erroneously displays "Home" for this waypoint, despite the "Home" waypoint in fact being #26.
Reinf At the time of writing, the map editor erroneously displays "Reinf" for this waypoint, despite the "Reinf" waypoint in fact being #27.
