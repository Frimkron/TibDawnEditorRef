Waypoints

Used for marking special points on the map, to be referenced by scripting. A scenario can have up to 27 waypoints defined, each attached to a cell on the map. See "Waypoints"

Name Description
P0-5 these waypoints are used as the possible player start positions in multiplayer
6-24 general-purpose waypoints
Flare marks the location at which a drop zone flare can be created. See "DZ at Z" trigger action type
Home marks where the camera will begin at the start of a singleplayer scenario
Reinf marks where reinforcements will arrive by air. See "Reinforce" trigger action type and "Air" source type.
